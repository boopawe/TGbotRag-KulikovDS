## О проекте

Telegram-бот, реализующий метод Retrieval-Augmented Generation (RAG) для ответов на вопросы пользователей. Бот использует лёгкую языковую модель TinyLlama-1.1B и векторную базу данных ChromaDB для хранения и поиска фактов.

### Почему TinyLlama?

- Лёгкая модель (1.1B параметров) - работает даже на обычном ноутбуке
- Быстрая генерация ответов (10-30 секунд на CPU)
- Полностью локальная работа - не требует интернета после установки
- Отличное соотношение скорости и качества для RAG-задач

## Основные возможности

- **Локальная LLM**: Генерация ответов с использованием модели TinyLlama-1.1B-Chat
- **Векторная база знаний**: Хранение и семантический поиск фактов через ChromaDB
- **RAG-система**: Автоматический поиск релевантного контекста в базе перед генерацией ответа
- **Управление данными**: Команды для добавления, просмотра и очистки базы знаний
- **Поддержка GPU**: Автоматическое использование видеокарты NVIDIA при наличии

## Содержимое базы знаний (астрофизика)

База данных содержит следующие факты:

1. Масса сверхмассивной чёрной дыры в центре Млечного Пути (Стрелец A*) составляет около 4 миллионов масс Солнца.
2. Самая быстро вращающаяся нейтронная звезда делает 716 оборотов в секунду.
3. Расстояние до ближайшей звезды Проксима Центавра — 4,24 световых года.
4. Возраст Вселенной составляет 13,8 миллиардов лет.
5. Гамма-всплески — самые мощные взрывы во Вселенной.

## Команды бота

| Команда | Описание | Пример |
|---------|----------|--------|
| `/start` | Запуск бота и приветствие | `/start` |
| `/add` | Добавить новый факт в базу | `/add Земля круглая` |
| `/get_all` | Показать все сохранённые факты | `/get_all` |
| `/generate` | Прямой запрос к ИИ (без контекста) | `/generate Почему небо голубое?` |
| `/rag` | Генерация ответа на основе базы знаний | `/rag Сколько лет вселенной?` |
| `/clear_db` | Очистить всю базу данных | `/clear_db` |
| `/help` | Показать список команд | `/help` |

## Установка и запуск

### 1. Клонирование репозитория

```
git clone https://github.com/ваш-username/TG_RAG_Bot
cd TG_RAG_Bot
```
### 2. Создание виртуального окружения
```
python -m venv venv
venv\Scripts\activate
```
### 3. Установка зависимостей
```
pip install -r requirements.txt
```

### 4. Установка PyTorch
```
# Для CUDA 11.8
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# Для CUDA 12.1
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```
### 5. Получение токена Telegram-бота
```
Перейдите к @BotFather в Telegram

Отправьте команду /newbot

Следуйте инструкциям для создания бота

Скопируйте полученный API-токен
```
### 6. Настройка окружения
```
BOT_KEY=ваш_токен_бота
```
### 7. Запуск бота
```
python run.py
```
