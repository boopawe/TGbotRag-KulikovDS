
## О проекте

Telegram‑бот, реализующий метод Retrieval‑Augmented Generation (RAG) для ответов на вопросы пользователей. Бот использует лёгкую языковую модель **TinyLlama‑1.1B‑Chat** и векторную базу данных **ChromaDB** для хранения и семантического поиска фактов.

### Почему TinyLlama?

- Лёгкая модель (1.1B параметров) — не требует мощных вычислений.  
- Быстрая генерация ответов (примерно 10–30 секунд на CPU).  
- Полностью локальная работа — не требует интернета после установки.  
- Отличное соотношение скорости и качества для RAG‑задач.

## Основные возможности

- **Локальная LLM**: генерация ответов с использованием модели TinyLlama‑1.1B‑Chat.  
- **Векторная база знаний**: хранение фактов и семантический поиск с помощью ChromaDB.  
- **RAG‑система**: автоматический поиск релевантного контекста в базе перед генерацией ответа.  
- **Управление данными**: команды для добавления, просмотра и очистки базы знаний.  
- **Поддержка GPU**: автоматическое использование видеокарты NVIDIA (CUDA), если она доступна.

## Содержимое базы знаний (астрофизика)

По умолчанию база содержит следующие факты:

1. Масса сверхмассивной чёрной дыры в центре Млечного Пути (Стрелец A*) составляет около 4 миллионов масс Солнца.
2. Самая быстро вращающаяся нейтронная звезда (пульсар PSR J1748-2446ad) делает 716 оборотов в секунду.
3. Расстояние до ближайшей к Солнцу звезды (Проксима Центавра) — 4,24 световых года.
4. Возраст Вселенной, согласно последним данным спутника Planck, составляет 13,8 миллиардов лет.
5. На Юпитере регулярно выпадают алмазные дожди.
6. Гамма-всплески — самые мощные взрывы во Вселенной, за секунду они выделяют столько же энергии, сколько Солнце за всю свою жизнь.

Список фактов можно расширять через команду `/add`.

## Структура проекта

Проект имеет следующую структуру:


TG_RAG_Bot:
- run.py # Главный файл бота (Telegram-логика)
- database.py # Класс для работы с ChromaDB
- llm.py # Класс для работы с моделью TinyLlama

Векторная база данных создаётся автоматически в папке `vector_store/` при первом запуске бота.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-username/TG_RAG_Bot
cd TG_RAG_Bot
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
venv\Scripts\activate
```

(На Linux/Mac: `source venv/bin/activate`)

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Установка PyTorch

#### Для GPU (NVIDIA, рекомендуется при наличии видеокарты):

CUDA 11.8:
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

CUDA 12.1:
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

#### Для CPU (если нет видеокарты NVIDIA):

```bash
pip install torch
```

### 5. Получение токена Telegram‑бота

1. Откройте Telegram и найдите пользователя `@BotFather`.  
2. Отправьте команду `/newbot` и следуйте инструкциям по созданию бота.  
3. Скопируйте полученный API‑токен (строка вида `123456789:ABC...`).

### 6. Настройка окружения

Создайте файл `.env` в корне проекта:

```env
BOT_KEY=ваш_токен_бота
```

### 7. Запуск бота

```bash
python run.py
```

При первом запуске:

- автоматически скачается модель **TinyLlama‑1.1B‑Chat** (около 2.2 ГБ);  
- создаётся папка `vector_store/` с векторной базой ChromaDB;  
- загружаются стартовые факты по астрофизике.

### **Сравнительная таблица**

| Аспект | LLM (базовая модель) | RAG (с базой знаний) |
|--------|----------------------|-----------------------|
| **Фактическая точность** | Почти нет фактов, много абстрактных рассуждений и галлюцинаций. | Устойчиво даёт правильное расстояние до Проксимы Центавра ≈4,24 световых года (≈40 трлн км). |
| **Релевантность** | Ответы часто уходят в философию, город, видимость и не отвечают напрямую на вопрос. | Стартует с прямого ответа на запрос, потом добавляет факты из базы. |
| **Структура ответа** | Обрывистые, несвязные фразы, теряется мысль. | Более структурировано, с попыткой расширить ответ деталями из базы знаний. |
| **Качество данных** | Галлюцинации и вымышленные конструкции вроде "до ближайшей звезды нет". | Часто вводит нерелевантные или неточные данные (например, о пульсаре PSR J1748‑2446ad и расстоянии 300 млн км — это не так, он лежит в другой системе). |
| **Полезность для пользователя** | Мало полезное: больше творческий поток, чем фактический ответ. | Значительно выше: пользователь сразу получает число и понимание, что ближайшая звезда — Проксима Центавра. |
